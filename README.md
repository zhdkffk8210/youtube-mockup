# 📺 YouTube Mockup Clone

HTML, CSS, JavaScript로 구현한 유튜브 메인 페이지 목업 프로젝트

---

## 🖥️ 프로젝트 소개

YouTube 메인 페이지 UI를 클론하여  
레이아웃 구성, 다크모드, 사이드바 토글, 인터랙션 기능 등을 구현한 프로젝트입니다.

---

## 🚀 주요 기능

### 1️⃣ 다크모드
- 버튼 클릭 시 다크모드 전환  
- localStorage를 활용하여 새로고침 후에도 상태 유지  
- CSS 변수 기반 테마 관리  

### 2️⃣ 사이드바 토글
- 햄버거 버튼 클릭 시 사이드바 축소/확장  
- collapsed 레이아웃 구현  
- 모바일 화면 대응  

### 3️⃣ 활성화(Active) UI 처리
- 사이드바 메뉴 클릭 시 active 변경  
- 카테고리 버튼 클릭 시 active 변경  
- 하나의 메뉴만 활성화 유지  

### 4️⃣ 반응형 레이아웃
- CSS Grid 사용  
- auto-fill + minmax 구조  
- 768px 이하 모바일 대응  

### 5️⃣ 썸네일 Hover 효과
- 이미지 확대(scale)  
- 자연스러운 transition 적용  

---

## 🛠️ 사용 기술

- HTML5  
- CSS3 (Grid, Flexbox, Variables)  
- Vanilla JavaScript  
- LocalStorage
